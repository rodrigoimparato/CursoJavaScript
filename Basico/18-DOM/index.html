DOM é a abreviação de Document Object Model.
Ele representa os objetos de uma página HTML em forma de árvore.
DOM é importanta pois através dele que acessamos os elementos dentro de uma página HTML.
Permite que programa e scripts acessem e atualizem dinamicamente o conteúdo, a estrutura e o estilo do documento.
